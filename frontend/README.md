# Freight Command Center

Realtime-first frontend prototype for a freight dispatching and broker intelligence platform.

## Stack

- Next.js App Router
- React + TypeScript
- Tailwind CSS
- Zustand
- TanStack Query
- TanStack Table
- TanStack Virtual
- Framer Motion
- Lucide Icons

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Current scope

- Frontend only
- Mock freight load data
- Mock WebSocket patch stream
- Virtualized realtime load table
- Intelligence drawer
- Search session tabs
- Filter strip
- Sidebar and topbar operational shell
- Activity/notification stream

Backend, database, scraping, automation workers, and real WebSocket transport are intentionally not implemented.
